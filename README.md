# Tuberculosis (TB) Data Analysis – Capstone Project

## Overview

This project was completed as part of a capstone assignment following training in R programming. The objective was to apply R skills to a real-world tuberculosis (TB) dataset, focusing on data cleaning, transformation, analysis, and visualization.

## Learning Outcome

This project reflects practical application of R in a public health context. Key skills developed include:

* Handling and cleaning complex, real-world datasets
* Creating derived variables for analytical purposes
* Conducting exploratory data analysis
* Developing visualizations using ggplot2
* Interpreting findings relevant to clinical and epidemiological contexts

## Dataset Description

The dataset contains patient-level TB information, including:

* Demographic details (age, gender, district)
* Clinical measurements (respiratory rate, BMI, haemoglobin, oxygen saturation)
* Symptom profiles and diagnostic information
* Risk stratification and management indicators

## Analysis Workflow

The analysis involved the following steps:

1. Data cleaning and preprocessing, including handling missing values
2. Creation of indicator variables to identify abnormal clinical parameters
3. Aggregation and summarization of patient-level data
4. Visualization of key variables to identify patterns and trends

## Key Analysis

A descending order column chart was generated to display the number of patients with abnormal values for:

* Respiratory rate
* BMI
* Haemoglobin
* Oxygen saturation

## Tools and Packages

* R
* RStudio
* ggplot2
* dplyr

## Conclusion

This capstone project demonstrates the application of R for analyzing real-world health data. It strengthened foundational data analysis skills and provided practical experience relevant to epidemiological research and public health data workflows.
